1. What Is a Security Deposit Refund?

A security deposit is money you pay to a landlord **before moving in** to cover potential damages. When you move out, the landlord must **return it within a set number of days** (varies by state).

Key idea

If the landlord **misses the deadline** or **deducts unreasonably**, you can **demand the full amount plus penalties**.

2. Who Can Claim the Refund?

  • The **tenant** (name on the lease)
  • **Joint tenants** (any one can claim)
  • **Heirs** (if tenant died)
  • **Assignees** (with written transfer)

Foreign nationals can claim without SSN—passport + lease is enough.

3. How to Get Your Security Deposit Back

  1. Document the unit (photos, video walkthrough)
  2. Leave a forwarding address in writing
  3. Wait for the state deadline (see table below)
  4. Send a demand letter if unpaid (template in PDF)
  5. File in small-claims court if ignored

4. Refund Deadlines by State (2025)

  • California: 21 days
  • Texas: 30 days
  • Florida: 15 days
  • New York: 14 days
  • Illinois: 30 days

FAQ

Can I claim if I moved abroad?

Yes. Provide an international forwarding address or US bank account for ACH.

What if the landlord deducted for “normal wear and tear”?

Illegal in most states. You can **demand the full deposit back**.

Is interest owed on late refunds?

Some states (e.g. NY, IL) require **interest**—PDF includes exact rates.
