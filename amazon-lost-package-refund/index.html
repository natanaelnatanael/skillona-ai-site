1. When You Can Legitimately Ask for a Refund or Replacement

Common valid situations:

  • Tracking says “Delivered” but you got nothing
  • Delivered to wrong address
  • Empty box / missing items
  • Transit damage
  • Severe delay or package stuck for days

2. Immediate Steps Before Contacting Amazon

  • Check order details and shipping address
  • Ask neighbors / check property
  • Take photos if item is damaged
Adsense block placeholder

3. Standard Refund / Replacement Flow

  • Orders → Select order
  • Problem with order
  • Pick issue → Refund / Replacement

4. Contacting Support (Chat or Phone)

Sample script

“Hello, my order [order number] shows delivered, but I didn’t receive it. I checked the area and neighbors. Can you help with a refund or replacement?”

5. Escalation Steps if Support Refuses

  • Ask for supervisor
  • Restate facts
  • If carrier photo is wrong, explain why
Important

Never lie. Amazon checks account history and may restrict users abusing refunds.

6. Limits and Account Safety

Too many claims → review. Genuine issues are safe.

7. FAQ – Short Answers

How long do I have to report?

Report as soon as possible — ideally within a few days.

Will Amazon ban me?

Only for excessive or suspicious claims.

Can I get refund if carrier says delivered?

Yes — explain clearly what happened.

Do I return damaged items?

Sometimes yes, sometimes no.
